# Technical references

This extension was written from scratch. No third-party source file is bundled.

The implementation was checked against:

- publicly observable, read-only SimCompanies GET responses;
- official SimCompanies articles describing the 2026 season calendar, scheduled season transitions, production events, Pumpkin seasonality, Summer weather, ice-cream decay, and Tree production mechanics;
- the AGPL-3.0 project `gangbaRuby/SimCompanies-Scripts` for cross-checking the traditional-retail equation and current game-data extraction strategy;
- the MIT-licensed project `Llyfrs/simcompanies-simulator` for production-field and SimcoTools VWAP schema handling;
- public SimcoTools API responses and public SimcoIntel snapshots for schema/reconciliation verification.

Community projects were used as technical references only. Their source files are not included in this distribution.
